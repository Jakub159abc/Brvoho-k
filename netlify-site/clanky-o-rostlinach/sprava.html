Správa stránky „Články o rostlinách“

Zpět na seznam článků

Jednotlivé články jsou samostatné HTML soubory ve složce projektu články html/rostliny/. Rozcestník s vyhledáváním se nepíše ručně – vygeneruje ho skript z názvů souborů a značky <title> v každém článku.

Přidání nebo úprava článku

  • Uložte nebo upravte soubor články html/rostliny/Váš název rostliny.html (název souboru = soubor na disku; titulek stránky vezměte z <title> v tom souboru).
  • Spusťte přegenerování rozcestníku (viz níže) a pak celý web pro náhled nebo deploy.

Přegenerovat rozcestník (seznam na stránce článků)

V kořeni repozitáře v příkazovém řádku:

python _generate_clanky_rostliny_index.py

Tím se přepíše site/clanky-o-rostlinach/index.html aktuálním seznamem odkazů.

V editoru Cursor / VS Code

  • Ctrl+Shift+B (nebo příkazovou paletu: „Tasks: Run Build Task“) – výchozí úloha sestaví celý netlify-site včetně přegenerování rozcestníku článků.
  • Nebo Ctrl+Shift+P → „Tasks: Run Task“ → Články: přegenerovat rozcestník rostlin – jen rozcestník bez kopírování celého webu.

Celý náhled jako na Netlify

Po změnách spusťte python _build_netlify_site.py a pak např. npx --yes serve netlify-site (viz soubor NAHLED-LOKALNE.txt).
